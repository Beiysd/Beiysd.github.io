# 自动聚焦

# 有两种模式:

1.autoFocus(React 中 F 是大写)属性使用： autoFocus='autofocus'

2.ref 使用：ref={(refs) => (this.input = refs)}

//js中声明
input = null;
if (this.input) {
  this.input.focus();
}
//页面使用
<div>
  <Input autoFocus="autofocus" />
  <Input ref={(refs) => (this.input = refs)} />
</div>;

区别：

autofocus： 初次渲染 Input 框不聚焦，手动聚焦后，再次进入可以自动聚焦(JS 原生中是生效的)

ref：初次/再次渲染都会自动聚焦，(**要保证 Input 框先渲染，方法后触发)
